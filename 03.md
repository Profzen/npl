## 🐍 INSTALLATION PYTHON 3.11 (RUNTIME PROJET)

### **1. Installation pyenv**

> pyenv permet de gérer Python 3.11 sans toucher au Python système (3.6.8).

```bash
sudo dnf install -y gcc make patch zlib-devel bzip2 bzip2-devel \
    readline-devel sqlite sqlite-devel openssl-devel tk-devel \
    libffi-devel xz-devel

curl https://pyenv.run | bash

cat >> ~/.bashrc << 'EOF'

# Pyenv
export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init --path)"
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"
EOF

source ~/.bashrc

pyenv --version
# pyenv 2.3.x
```

---

## 🌐 CONFIGURATION NGINX

### **1. Installer Nginx**

```bash
sudo dnf install -y nginx

nginx -v
# nginx version: nginx/1.20.x
```

---

### **2. Créer la configuration backend**

```bash
sudo vi /etc/nginx/conf.d/smart_transfer_back.conf
```

**Contenu :**

```nginx
upstream smart_transfer_backend {
    server 127.0.0.1:8000;
}

server {
    listen 80;
    server_name api.smart-transfer.local;

    access_log /var/log/nginx/smart_transfer_backend_access.log;
    error_log /var/log/nginx/smart_transfer_backend_error.log;

    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;

    add_header 'Access-Control-Allow-Origin' 'http://localhost:3000' always;
    add_header 'Access-Control-Allow-Credentials' 'true' always;
    add_header 'Access-Control-Allow-Methods' 'GET, POST, PUT, DELETE, OPTIONS' always;
    add_header 'Access-Control-Allow-Headers' 'Content-Type, Authorization, X-Idempotency-Key' always;

    if ($request_method = 'OPTIONS') {
        return 204;
    }

    location /api {
        proxy_pass http://smart_transfer_backend;

        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;

        proxy_pass_header Set-Cookie;

        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }

    location /health {
        proxy_pass http://smart_transfer_backend/health;
    }
}
```
